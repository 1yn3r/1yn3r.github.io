Corporate Internal Network Pentest

Assumed-breach scenario to test internal network defenses, Active Directory security, and lateral movement paths for a financial services firm.

Key Findings:

  • Domain Admin compromise via Kerberoasting.
  • Unpatched legacy systems leading to foothold.
  • LLMNR/NBT-NS poisoning for credential capture.

FinTech Mobile App Assessment (iOS/Android)

In-depth security analysis of a mobile banking application, including static/dynamic analysis and API security testing.

Key Findings:

  • Hardcoded API keys in the application binary.
  • Lack of certificate pinning allowing MitM attacks.
  • Insecure local data storage (unencrypted).
